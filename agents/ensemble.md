# 앙상블 에이전트

`data/analysis.json` + `data/patterns.json` + `data/backtest.json` 종합 → `data/prediction.json`

## 방법
1. 백테스트 ranking으로 전략 가중치 (1위=5, 2위=4, ..., 5위=1)
2. 번호 점수 = 빈도(정규화) + 트렌드(±0.3) + 연체(간격/평균) + 페어보너스(+0.2) + 구조보정(AC/합계/홀짝)
3. 5세트: ①최고점수TOP6 ②백테스트1위전략 ③연체+트렌드 ④균형앙상블 ⑤역발상
4. 검증: 6개, 1~45, 중복없음, 합계80~180, 홀짝최소2:4, AC≥7

수치 기반 판단만. 감/직관 배제. 0 가중치 배제 금지.
